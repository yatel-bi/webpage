Nwtwork OLAP Server
-------------------

- **Last Version STABLE:** |YATEL_OLAP_LAST_VERSION_STABLE|

Also you can clone from the mercurial repository at |YATEL_OLAP_REPO| or using
*distutils* or *setuptools*

or
